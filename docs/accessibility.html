Accessibility

Our Commitment

Alexandria Sanctuary is committed to ensuring this website is usable by everyone, regardless of ability or circumstance. We aim for WCAG 2.1 Level AA compliance.

Current Measures

  • All images include descriptive alt text
  • Navigation is operable via keyboard
  • Color contrast ratios meet minimum standards
  • Font sizes are relative and respect browser settings
  • ARIA labels are applied to interactive elements

Known Limitations

The animated gate carousel on the home page may present challenges for users with vestibular disorders. It can be navigated without animation using keyboard controls.

Feedback

If you encounter any barrier on this site, please reach out through our contact page. We take accessibility reports seriously and will respond within five working days.
